# <img src="img/README8.png" height="28" style="vertical-align: -5px;"> Quick Explorer

**Quick Explorer** is a high-performance, modern file manager for Windows built with **Tauri**, **Rust**, and **React**. Designed for speed, stability, and a premium user experience.

Developed with ❤️ using **Antigravity**.

---

## ✨ Core Features

### 📂 Advanced Tab System
- **Dynamic Multi-tabbing**: Open and manage multiple locations with ease.
- **Stay in Flow**: Smart navigation that remembers your position and restores your view perfectly.

### 🛡️ Rock-Solid Windows Integration
- **Native Drag & Drop**: Move files between Quick Explorer and any other Windows app with a high-fidelity experience, including real thumbnails and file count badges.
- **Instant Shell Previews**: High-speed thumbnails for all your images and videos, exactly as you see them in Windows.
- **Source Intelligence**: Visual indicators let you know if a preview is coming from Windows or generated by our engine.

### 📦 Native Archive Support
- **One-Click Extraction**: Built-in support for **ZIP** and **7Z** archives. No extra software needed.
- **Visual Progress**: See exactly how much time is left with real-time progress bars in the app and on your Windows Taskbar.

### 🌐 Intelligent Localization (i18n)
- **Speaks Your Language**: Automatic detection of English or Spanish based on your system, with seamless manual switching.
- **Full Translation**: From file types to drive labels, every part of the interface feels native.

### 🔍 QuickPreview Overlay
- **Instant Discovery**: Tap the `Spacebar` or use the **Middle Mouse Button** to instantly peek inside files without opening them.
- **Intuitive Controls**: 
    - **Universal Zoom**: Double left-click to zoom in and double right-click to zoom out of photos and videos.
    - **Smart Navigation**: Quickly cycle through files while the viewer automatically skips empty space.

### 🎨 Modern Adaptive UI
- **Smart Toolbar**: The interface automatically adjusts its layout based on window size, keeping your workspace clean and professional.
- **Folder Intelligence**: Instant folder size calculations that are remembered for your next visit.

### ⚡ Performance First
- **Always Fluid**: Everything happens in the background, so the interface never freezes, no matter how many files you have.
- **Zero-Flicker Probing**: Media information is gathered silently, ensuring a smooth and uninterrupted experience.

## 🛠️ Technology Stack

- **Backend**: [Rust](https://www.rust-lang.org/) + [Tauri v2](https://v2.tauri.app/)
- **Frontend**: [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/) + [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS with modern aesthetics.
- **IDE**: [Antigravity IDE](https://antigravity.google)

## 🚀 Getting Started

### Installation (Recommended)

The easiest way to install **Quick Explorer** on Windows is via [Winget](https://learn.microsoft.com/en-us/windows/package-manager/winget/):

```bash
winget install biondi53.QuickExplorer
```
